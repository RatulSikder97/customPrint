# CUSTOM PRINT

**Print a string n times without loop**

## Code Snippet

```javascript
function customPrint(n, message) {
  console.log(Array(n + 1).join(message + "\n"));
}
```

### Simple I/O

- **_Input_**

```javascript
//INPUT
customPrint(7, "Hello, World!");
```

- **_Output_**

```
Hello, World!
Hello, World!
Hello, World!
Hello, World!
Hello, World!
Hello, World!
Hello, World!
```
